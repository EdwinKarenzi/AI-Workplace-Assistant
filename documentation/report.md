# AI Workplace Incident Assistant

## Projektbeskrivning
Detta projekt är en AI-baserad webbapplikation som analyserar arbetsplatsincidenter och genererar säkerhetsrapporter automatiskt med hjälp av Generativ AI.

## Syfte
Syftet med projektet är att skapa en praktisk AI-lösning för APL och arbetsplatser där användare snabbt kan analysera incidenter och få rekommenderade säkerhetsåtgärder.

## Funktioner
- Incidentanalys
- Riskbedömning
- AI-genererad incidentrapport
- Rekommenderade åtgärder
- Säkerhetschecklista

## Teknologier
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- OpenRouter API
- Generativ AI

## Systemdesign
Frontend skickar användarens incident till backend via API-anrop. Backend kommunicerar med AI-modellen genom OpenRouter API och returnerar AI-genererat svar till användaren.

## GenAI Integration
Projektet använder Generativ AI för att:
- analysera incidenter
- identifiera risknivå
- generera rapporter
- föreslå säkerhetsåtgärder

## AWS Deployment
Applikationen hostades på AWS EC2 och gjordes tillgänglig via publik IP-adress.
